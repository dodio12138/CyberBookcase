# 赛博书架
#### 介绍
这是一个用于展示年度读书清单的生成器。可以按照年份生成一个虚拟书架。

#### 使用
通过配置 `booklist.json` 文件增加书架的书籍。下面是一些参数说明：
    `title` : 书籍名称
    `author` : 作者
    `titleColor` : 书名颜色
    `pages` : 书籍页数，该参数影响生成书籍的厚度
    `color` : 书籍颜色
    `format` : 书籍开本，影响书籍的高度
    `packaging` : 书籍的装订方式。设置 `P` 为软装， `H` 为硬装。影响书籍的展示效果。
    `classifications` : 书籍分类
